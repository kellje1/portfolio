# Portfolio Website — Web Design Document

## 1. Project Overview
The purpose of my portfolio is to show employers how capable I'am in becoming a web developer. 

## 2. Target Audience
This website is for employers 

## 3. Content Strategy
The home page will have a brief introduction about me and some basic questions that employers will ask me. 
The about me page will explain my journey in majoring in CPIS and how I wanted to become a web developer. 
The resume page will include my resume. 
The projects page will show all of the projects I've done. 
The contact me page will show my contact information.  

## 4. Information Organization
In the navigation bar, sections are included, such as the home, about me, resume, projects, and contact me pages, which they will give access.
I've organized the images into boxes and cards 

## 5. Visual Design
I've used lightyellow for the background color and lightsalmon as the background color for the navigation and footer bars. 
For the words in the nav and footer bars, I've used black for readability. 
For typography, I've used Franklin Gothic Medium, Arial Narrow. 
For the layout, I've chose to put the images next and above to the text for creativity. 
I've decided to make the background for the image cards pink.
