# Portfolio Website — Web Design Document

## Project Overview
The purpose of my portfolio is to showcase my skils and creativity and show employers how capable I'am in becoming a web developer. I built it because I want employers to see my creative side and learn more about my goals. 

## Target Audience
This website is for employers who are looking to hire new web developers, recruiters reviewing portfolios, and anyone who is interested in my background. 

## Content Strategy
The home page will have a brief introduction about me and some basic questions that employers will ask me. 
The about me page will explain my journey in majoring in CPIS and how I became interested in web development.
The resume page will include my resume. 
The projects page will show all of the projects I've done. 
The contact me page will show my contact information for employers and visitors. 

## Information Organization
In the navigation bar, sections include the home, about me, resume, projects, and contact me pages, which they will give access when switching to one section to another. 
I've organized the images into boxes and cards to make the layout consistent. 

## Visual Design
I've used lightyellow for the background color and lightsalmon as the background color for the navigation and footer bars. 
For the words in the nav and footer bars, I've used black for readability. 
For typography, I've used Franklin Gothic Medium, Arial Narrow. 
For the layout, I've chose to put the images beside and above to the text for creativity. I used pink backgrounds for the image cards to make them stand out.
